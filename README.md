# Instalacao e como Rodar ‍💻

Instale o Node Package Manager(NPM) em sua maaquina, depois da instalacao, efetue os comandos abaixo:

```cmd

npm install
npm run dev

```

# Utilizar testes 🧪

Depois de voce ter completado a fase de [Instalacao](#instalattion-and-run) basta voce rodar os comandos a seguir no terminal certificando-se de que esta no repositorio raiz:

```cmd
  npm run test
```
